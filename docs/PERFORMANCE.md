# SafePrompt Guard — 성능 측정 결과

> 자동 생성: 2026-05-20 06:45 UTC  
> 환경: Linux-5.15.0-139-generic-x86_64-with-glibc2.31 · Python 3.10.16  
> Gemma: 미연결 (정규식+규칙만)  
> API 입력 상한: **1,048,576자** (약 1.00MiB)

## 목표 (과제 정량 지표)

- **1MB 이하** 텍스트 기준 **평균 5초(5000ms) 이내** 분석 결과 출력
- 실제 API는 Pydantic `max_length=1048576` 로 **최대 1,048,576자**까지 허용

## 측정 방법

- `run_scan()` 직접 호출 (HTTP 업로드·네트워크 오버헤드 제외)
- 샘플: 일반 설명 문장에 설정·API Key·JWT·DB URL을 섞은 합성 텍스트
- 각 크기·모드별 **3회** 반복 후 평균/최소/최대(ms)
- Gemma 생략: `--regex-only` 모드

## 요약

- 정규식+규칙만: 5/5 구간이 평균 5초 이내
- Gemma 미연결 — 3차 문맥 분석 시간은 별도 측정 필요
- 최대 측정 크기 1,048,576자 = API 상한 근처 (과제 1MB 목표에 대응)
